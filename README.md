# THE TIME BEING  
*A Story For All Ages*  

Official website for **THE TIME BEING**, a literary work by **Etienne de Montebello**.  
This site presents the book as an eBook release, alongside related screenplays and supporting material.

---

## About this site

This website is intentionally minimal and stable.  
It is designed as both:

- a **reading space**  
- and a **presentation space** for the work  

The opening page introduces the conceptual atmosphere of the story.  
Page 2 presents the content in a clear, editorial layout focused on accessibility and legibility.

The site is intended to remain unchanged for an initial evaluation period.
